## MODIFIED Requirements

### Requirement: Individual projections display layout
The system shall display husband and spouse individual projections horizontally side-by-side for easier comparison.

#### Scenario: Desktop viewport display
- **WHEN** user views the two-person retirement calculator on a desktop screen (≥768px width)
- **THEN** husband's projection appears in the left column and spouse's projection appears in the right column

#### Scenario: Mobile viewport display
- **WHEN** user views the two-person retirement calculator on a mobile device (<768px width)
- **THEN** projections stack vertically with husband above spouse for better readability
